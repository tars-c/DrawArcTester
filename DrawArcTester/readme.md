# DrawArcTester Program
## 요구사항
- \.Net Framework 4.5

## 설명
이 프로그램은 Graphics.DrawArc 함수가 요구하는 매개변수 값을 눈으로 확인할 수 있습니다.
추가로 폼 에디터에서 스크롤바(vertical)를 추가하여 스크롤을 처리하는 연산코드를 포함하고 있습니다. (마우스 휠 개별 처리)

간단한 테스트 목적으로 만들어진 것이기에 오류가 있을 수 있습니다.
